# SuperMarket Management System

## Project Overview
This project is a SQL-based SuperMarket Management System developed using MySQL.

## Features
- Customer Management
- Product Management
- Orders Management
- CRUD Operations
- Primary Key & Foreign Key
- INNER JOIN
- GROUP BY
- HAVING
- Views
- Transactions

## Technologies Used
- MySQL
- SQL
- MySQL Workbench

## Database Tables
- Customer
- Product
- Orders
